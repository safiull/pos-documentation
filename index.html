Installing equationPOS

Server requirements

Make sure your server meets the following requirements:

  1. PHP >= 7.2
  2. OpenSSL PHP Extension
  3. PDO PHP Extension
  4. Mbstring PHP Extension
  5. Tokenizer PHP Extension
  6. XML PHP Extension
  7. cURL PHP Extension
  8. Zip PHP Extension
  9. GD PHP Extension

Server Recommendation

  • Equation POS can be installed in Localhost.
  • If you want to make it accessible online- We recommend using Cloud host – as it provides better flexibility or control of the server. You can use AWS or Digital Ocean or search for some others. Our demo site is set up on the Digital Ocean.
  • You can also use Shared hosting. Like for example BigRocks, NameCheap, GoDaddy etc.

STEPS TO INSTALLING POS SYSTEM

  1. After downloading, extract the file onto your desktop using a zip software (like winzip or winrar or something else ) as shown below.
  2. In case you want to install the system locally, here are the steps:
    • 2.1 Download & Install a server like WAMP(for Windows) or XAMP(for windows or Linux) or something else as per your preference having Apache or nginx server.
    • 2.2 Go to the htdocs folder. htdocs folder is commonly present inside the xamp/htdocs.
    • 2.3 Copy the codebase folder (For example EquationPOS-V1.0.0) present inside the unzipped file in step 1.
    • 2.4 Rename the folder to EquationPOS. (For example: EquationPOS-V1.0.0 to EquationPOS)
    • 2.5 Now open http://localhost/EquationPOS/install or http://localhost/EquationPOS/public/install Here you will see a screen with “Doing some pre-installation checkup…”
    • 2.6 Make sure all checks are shows as Green-Tick. If there is something with Red-Cross then fix it and refresh the page to check again. After all Green-Tick click on “Go to Next Step”
    • 2.7 Follow the installation instructions as displayed in the screen.
    • 2.8 After all installation is done, Go inside public folder and delete the install folder present.
  3. In case you want to install the system in hosting, here are the steps:
    • 3.0 Make sure the URL of application is like www.yourwebsite.com or any-subdomain.yourwebsite.com for better experience. Installing in folders like www.yourwebsite.com/any-folder-name/pos is not recommended.
    • 3.1 Upload the extracted POS zip file (all files & folders inside EquationPOS-V1.0.0) in the server document root.
      • To upload the file you can use FTP client like filezilla (https://filezilla-project.org/)
      • Or you can upload it directly by logging into your cPanel (or direct admin) & going to file manager.
    • 3.3 Open your website http://your-pos-website.com/install or http://your-pos-website.com/public/install.
    • 3.4 Make sure all shows a Green-Tick. If there is something with Red-Cross then fix it and refresh the page to check again. After all Green-Tick click on “Go to Next Step”.
    • 3.5 Follow the installation instructions as displayed in screen.
    • 3.6 You will see the installation page as given below or if shows some error then check debugging errors notes given below.
    • 3.7 After all installation is done, Go inside public folder and delete the install folder present.

Product Manager

Add New Product

How to add a new product ? 🙄
Step 01: Go to – Settings > Add Products.

1) Add Product Name: Add Product Name in this field. This field is required. The Product Name field must be at least 3 characters.

2) Unit: Add product Unit name in this field, Ex: Piece, Dozen etc. This field is required. The product Unit field must be at least 2 characters.

NOTE: You can add a new unit from this page. Just clik on icon.

3) Category: Add Category of product in this field. This field is NOT required. The Category field must be at least 2 characters. Sub-Category can be added under the parent Category.

NOTE: You can add a new category from this page. Just clik on icon.

4) SKU: Add a unique SKU(Stock-keeping unit) which will be used to identify the product. This field is NOT required.

5) Barcode Type: Select Barcode Type from this field. This field is NOT required.

6) Brand: Select Brand of product from this field. This field is NOT required. The Brand field must be at least 2 characters.

NOTE: You can add a new brand from this page. Just clik on icon.

7) Weight: Add Weight of product in this field. This field is NOT required.

8) Inventory Control: Check Inventory Control field if prevent selling when Product is out of stock and get Alert Stock Quantity. This field is NOT required.

9) Alert Stock Quantity: Fill up Alert Stock Quantity number in this field. User will get an alert once the Stock Quantity will be same to this number. This field is required if "Inventory Control" is checked.

10) Cost Price (Cost of Goods Sold): Add Cost Price of Product in this field. This field is NOT required. This Price is without VAT. But if this field is filled up than this price will be shown in new Purchase Order's price field.

11) Add VAT: Add VAT of Product Purchased in this field. If Product has VAT then checked the check box to put the VAT percentage. This field is NOT required.

12) Net Cost Price: Net Cost Price will be automatically calculated after VAT(if have) from Cost Price. This field is NOT required.

13) Profit Margin(%): This field will show the Percentage of Profit Margin. Also user can set Profit Margin by this field. This field is NOT required.

14) Gross Selling Price: User can set Gross Selling Price in this field or it will be automatically calculated based on Net Cost Price & Profit Margin(%). This price will be shown in POS product list as actual price of product if this has no Discount. This field is NOT required.

15) Add Discount: User can set individual Product Discount in this field. To add Discount first checked the check box than select the Discount type (Fixed or Percentage) from dropdown. It will effect the Gross Selling Price in the POS product list. This field is NOT required.

16) Selling Price After Discount: If product has discount than this field will show the Product selling price after dedcuting the discount. And this price will be shown in the POS product list as actual price of product and "Gross Selling Price" will be crossed. This field is NOT required.

17) Add VAT: Add VAT of Selling Product in this field. If Product has VAT then checked the check box to put the VAT percentage. This field is NOT required.

18) Net Selling Price: This price will be automatically calculated after Discount and VAT on Gross Selling Price. This field is NOT required.

19) Description: Add Product details descripton in this field. This description will be shown on E-commerce Product page. This field is NOT required.

20) Product Short Description: Add Product Product Short Description in this field. This field is NOT required.

21) Upload Image: Upload image of product in this field. Recommended Image Dimension : 500px X 500px and Size : Max 1 MB. This field is NOT required.

Step 02: Click on -> Create new product.

Add New Purchase

How to add a new purchase order? 🤔

Please follow our instruction. I hope you can do it easily!

Let's start 🤓
Step 01: Go to – Inventory > Add Purchase Order.

1) Supplier Name: Select Supplier Name of Product from this field. This field is required. Only Products which are added under this supplier will be shown in "Product Search" field.

2) Purchase Order Number: Put a Purchase Order Number in this field. This field is required.

3) Date: Select Purchase Order date in this field. This field is required.

4) Add product( Which is you want buy ).



A) Add Products from selected supplier in this field. This field is required.

B) This column will show the Serial Number of Products.

C) This column will show the Name of Products.

D) User can add Order Quantity of individual Product in this field. This field is required.

E) This Unit Cost of individual Product is generated from "Cost Price (Cost of Goods Sold)" field of "Add New Product" page, but user can add/change it here. This field is required.

F) This column will show the Total Cost (Order Quantity x Unit Cost) of individual Product.

G) User can set individual Product Discount in this field. This field is NOT required.

H) This column will show the Discount Amount of individual Product.

I) This column will show the Cost After Discount of individual Product.

J) User can set VAT Percent of individual Product in this field. This field is NOT required.

K) This column will show the VAT Amount of individual Product.

L) This column will show the Cost After VAT of individual Product.

M) User can set the Expiry Date of individual Product in this field. This field is NOT required.

N) User can set the Lot / Batch No. of individual Product in this field. This field is NOT required.

O) User can remove an individual Product from this action column.

P) This column will show the Total Amount of all selected Products after DEDUCTING individual Product Discount and ADDING individual Product VAT.

Q) User can set bulk Discount on all Products in this field. This field is NOT required.

R) This column will show the Purchase Total of all selected Products after DEDUCTING individual Product Discount, ADDING individual Product VAT and DEDUCTING bulk Product Discount.

A) Add Amount which user want to pay to supplier in this field. This field is NOT required.

B) Select the payment method from in this field.

C) Select the account number of payment method from in this field.

D) Add note of payment in this field. This field is NOT required.

F) Make draft of the Purchase Order by pressing this button.

POS



A) User can search product from this field.

B) All Products will appear on left side of POS page.

C) Products which are "Out of Stock" will appear in RED box. Products which are "Out of Stock" can not be sold until it's "Inventory Control" option is uncheked in product page.

D) User can search customers from this field.

E) User can add new customers from this add button.

F) All selected products from left will be added in this cart area.

G) Product quantity can be increase/decrease from these buttons.

H) This is "Gross Selling Price" from product page without Discount & VAT.

I) This is "Selling Price After Discount" from product page with Discount & without VAT.

J) Total price of all selected products will be shown here with Discount(If have) & without VAT.

K) Total discount amount will be shown here and user can set total discount on all products from here, this total discount will only apply to those single products which has no discount or has discount but less than this total discout. M) Total Shipping/Delivery amount will be shown here and user can set total Shipping/Delivery cost from here.

N) User can Park a sell for temporary by this button.

O) User can Cancel a sell by this button.
